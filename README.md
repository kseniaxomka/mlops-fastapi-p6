Обзор проекта
Веб-сервис на базе FastAPI для детекции краев на изображениях с использованием алгоритма Canny из OpenCV.
Сервис принимает изображение, находит на нем контуры объектов и возвращает обработанное изображение с выделенными краями (красный цвет).

Основные функции:
1)Прием изображений через HTTP-запросы
2)Детекция краев с помощью алгоритма Canny
3)Визуализация результатов с наложением красных контуров
4)REST API с автоматической документацией Swagger
5)Потоковая передача результатов в формате JPEG

Класс EdgeDetector - для детекции краев на изображениях.

Метод detect_edges:
-Конвертация в оттенки серого
-Размытие по Гауссу, чтобы находить более значимые детали
-Детекция краев
-Выходные данные

Метод overlay_edges:
-Создание цветной маски краев
-Наложение краев на оригинал

FastAPI эндпоинт /detect-edges:
-Валидация файла
-Обработка изображения
-Детекция и визуализация
-Подготовка ответа

pip install -r requirements.txt
python main.py
http://localhost:8000
